# Glossaire

## API autorisée

Option de champ permettant son exposition par l'API, les statistiques et certaines
sorties de plugins.

## Article Joomla

Contenu Joomla éventuellement créé et lié à un enregistrement ContentBuilder NG.

## Backend

Administration Joomla.

## BreezingForms / BF

Composant de formulaires pouvant servir de source à une vue ContentBuilder NG.

## CB

Abréviation historique de ContentBuilder.

## CBNG

Abréviation de ContentBuilder NG.

## Champ / élément

Définition d'une donnée de la source, avec des options de liste, recherche, édition,
API et rendu.

## Collation / interclassement

Règles utilisées par MySQL ou MariaDB pour comparer et trier les textes.

## Datatable Sync

Action d'administration qui synchronise la définition d'un stockage avec sa table.

## Détail

Page affichant un enregistrement selon le template de détail.

## Éditer par type

Option qui délègue l'éditeur au composant source, par exemple BreezingForms.

## Enregistrement / record

Une instance de données, généralement une ligne de la source, identifiée par un
`record_id`.

## État de liste

État métier personnalisé associé à un enregistrement, distinct de sa publication.

## Frontend

Partie publique ou espace utilisateur du site Joomla.

## Groupe Joomla

Groupe d'utilisateurs utilisé par l'ACL et l'héritage des permissions.

## Layout

Présentation Joomla choisie par un élément de menu, par exemple liste standard,
cartes, tuiles ou compact.

## List Access

Permission propre à ContentBuilder NG autorisant l'accès à la liste.

## Own / propre

Règle limitant une action aux enregistrements appartenant à l'utilisateur.

## Plugin de contenu

Plugin Joomla transformant une balise dans un template ou un article, par exemple
`CBStats`, `CBRating`, `CBDownload` ou `CBImageScale`.

## Préparation

Code PHP exécuté avant le rendu d'un template détail ou édition.

## Publication

État contrôlant la visibilité d'une vue ou d'un enregistrement. Les dates de début et
de fin peuvent aussi intervenir.

## REPAIR DB

Workflow de réparation proposé dans l'écran À propos après un audit.

## Sparse fieldset

Paramètre `fields[ressource]` limitant les champs et ressources d'une réponse API.

## Stats

Permission et endpoint donnant accès aux statistiques d'une vue.

## Stockage interne

Stockage dont la table préfixée Joomla est créée et gérée par ContentBuilder NG.

## Stockage externe / bytable

Stockage relié à une table SQL existante.

## Template

Code HTML/PHP configuré dans une vue pour afficher ou éditer les données.

## Thème

Plugin fournissant styles, JavaScript et exemples de templates.

## Vue

Configuration fonctionnelle qui relie une source à une liste, un détail, une édition,
des permissions et des comportements.

## Vérification

Étape facultative conditionnant certaines permissions ou workflows, avec des plugins
comme Pass-Through ou PayPal.
